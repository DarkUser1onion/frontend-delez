# Проблема: Отсутствует проверка доступности API и curl в скриптах релиза

## Описание
При запуске `make release-check` нет проверки, что:
- curl установлен в системе
- API https://api.delez-repo.ru/v1/health отвечает

Из-за этого релиз может быть собран даже при нерабочем бэкенде.

## Шаги воспроизведения
1. Запустить `make release-check` (или `scripts\release-check.bat`)
2. Сборка проходит успешно даже если API недоступен или curl отсутствует.

## Ожидаемый результат
Перед сборкой должны быть проверены:
- наличие curl (с понятной ошибкой)
- ответ API (200 OK)

## Фактический результат
Проверок нет.

## Окружение
- ОС: Linux / Windows
- Версия Node: 20
- Проект: Delёз v0.1.0
