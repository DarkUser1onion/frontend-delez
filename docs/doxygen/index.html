Delёz Frontend — документация

Что это такое

Delёz Frontend — SPA на базе React 19 + Vite, реализующий пользовательский интерфейс персонального ИИ-ассистента Delёz. Приложение подключается к LangGraph Server API и предоставляет чат-интерфейс, историю диалогов, 3D-визуализацию ризомы и страницы управления событиями, целями и экспериментами.

Название и концепция вдохновлены философией ризомы Жиля Делёза: вместо линейного дневника пользователь получает живой граф, где события, эмоции и выводы связаны между собой.

Production-домен: https://delez.tech

Провайдеры состояния

StreamProvider

Файл: src/providers/Stream.tsx

Создаёт потоковую сессию через useStream из @langchain/langgraph-sdk/react. Управляет конфигурацией подключения (apiUrl, assistantId, apiKey), обрабатывает кастомные UI-события и синхронизирует threadId с URL.

ThreadProvider

Файл: src/providers/Thread.tsx

Загружает список диалогов через chatApi.getConversations() и адаптирует формат нашего API к формату Thread из LangGraph SDK. Предоставляет getThreads(), threads, threadsLoading через контекст.

Компоненты чата

Thread (основной layout)

Файл: src/components/thread/index.tsx

Двухпанельный layout: боковая история диалогов + основная область сообщений. Обрабатывает отправку сообщений, голосовой ввод, подсказки от LLM, регенерацию ответов и автопрокрутку.

Сообщения

  • src/components/thread/messages/human.tsx — сообщение пользователя с редактированием
  • src/components/thread/messages/ai.tsx — ответ ассистента с Markdown, tool calls, interrupts
  • src/components/thread/messages/tool-calls.tsx — детальный вывод вызовов инструментов
  • src/components/thread/messages/reactions.tsx — реакции на сообщения

История диалогов

Файл: src/components/thread/history/index.tsx

Загружает и отображает список диалогов. На больших экранах — фиксированная боковая панель, на мобильных — выдвижной Sheet (Radix UI).

API-клиент

Файл: src/lib/api-client.ts

Централизованный HTTP-клиент с поддержкой Bearer-токена из localStorage и Basic Auth для прокси. Содержит специализированные объекты:

Объект Назначение
authApi Вход, регистрация, сброс пароля, верификация email
chatApi Диалоги, сообщения, реакции, контекст треда
graphApi Ризома (граф), поиск по графу
entriesApi Записи дневника и их анализ
goalsApi Цели и связанные записи
experimentsApi Эксперименты и метрики интенсивности
hintsApi Динамические подсказки от LLM
